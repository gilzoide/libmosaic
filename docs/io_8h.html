Detailed Description

Mosaic's input/output definitions, for handling saving and loading from files.

The .mosi file has a header with the mosaic dimensions, the asc art itself and it's attributes.

Note
A .mosi file is hybrid. The dimensions and MOSAIC::mosaic are text, so they can be viewed in any text editor. The MOSAIC::attr part is binary, so expect some weird stuff while oppening it as text.
Note also that there are more than one way of keeping the binary part, from uncompressed, to compressed and no binary at all. There's a separator after the text part of the file, which is SEPARATOR and one of mos_attr_storage_fmt.
