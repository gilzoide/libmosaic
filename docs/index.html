Mosaic
The Mosaic ASC Art library
libmosaic

An open source ASC Art intended library.

Why ASC Art? One doesn't question art, you just feel it.

Optional runtime dependencies: zlib.

Building

Using CMake: 

$ mkdir build
$ cd build
$ cmake ..
$ make

Install

From build directory: 

$ make install
