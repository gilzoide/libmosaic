libmosaic
=========
An open source ASC Art intended library.

__Why ASC Art?__ One doesn't question art, you just feel it.

Optional runtime dependencies: [zlib](http://www.zlib.net/).


Building
--------
Using [CMake](https://cmake.org/):

	$ mkdir build
	$ cd build
	$ cmake ..
	$ make


Install
-------
From build directory:

	$ make install
